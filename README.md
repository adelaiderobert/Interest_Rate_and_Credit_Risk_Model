# Interest_Rate_and_Credit_Risk_Model

This repository contains two academic projects completed for the **Interest Rate and Credit Risk Models** course (EPFL, Fall 2025).

The projects focus on key topics in quantitative finance, including:

- zero-coupon bond pricing
- forward rates and FRA valuation
- yield curve modeling
- Nelson-Siegel / Nelson-Siegel-Svensson calibration
- Gaussian HJM modeling
- bond volatility dynamics
- simulation and derivative pricing

## Project 1 — Yield Curves, FRA Valuation, and Term Structure Modeling
Overview

Project 1 covers core fixed-income concepts and term structure analysis.

Main topics include:

zero-coupon bond pricing from continuously compounded yields
simple forward rate extraction
FRA valuation
sensitivity to parallel and non-parallel yield curve shifts
duration and convexity interpretation
calibration of yield curves using Nelson-Siegel and Nelson-Siegel-Svensson models

## Project 2 — Gaussian HJM Model and Interest Rate Derivatives
Overview

Project 2 focuses on more advanced continuous-time interest rate modeling.

Main topics include:

Gaussian Heath-Jarrow-Morton (HJM) framework
derivation of the arbitrage-free drift condition
bond price volatility under the risk-neutral measure
numerical evaluation of drift and volatility term structures
stochastic process analysis
simulation of forward rates and bond prices
caplet pricing
Black implied volatility analysis
parameter sensitivity and model validation
